📱 工时统计 APK 构建指南

⚠️ 注意： 此方法需要您有 GitHub 账户

最简单的方法：

步骤 1： 打开这个链接
点击这里开始构建
步骤 2： 在打开的页面：
  • 点击 "Create a new workflow"
  • 或者点击 "New workflow"
步骤 3： 复制下面的代码：
name: Build APK on: [push] jobs: build: runs-on: ubuntu-latest steps: - uses: actions/checkout@v4 - uses: actions/setup-node@v4 with: node-version: '20' - uses: actions/setup-java@v4 with: distribution: 'temurin' java-version: '17' - run: | mkdir -p client cd client echo '{"name":"workhours","version":"1.0.0","main":"expo-router/entry"}' > package.json echo '{"expo":{"name":"工时统计","slug":"workhours","version":"1.0.0","android":{"package":"com.workhours.app"}}}' > app.json mkdir -p app assets echo "placeholder" > assets/icon.png echo 'import {Text,View} from "react-native";export default function(){return 工时统计}' > app/index.tsx echo 'import {Stack} from "expo-router";export default function(){return }' > app/_layout.tsx - run: npm install -g @expo/cli && cd client && npx expo prebuild --platform android --clean - run: cd client/android && chmod +x gradlew && ./gradlew assembleDebug --no-daemon - uses: actions/upload-artifact@v4 with: name: APK path: client/android/app/build/outputs/apk/debug/app-debug.apk
步骤 4： 粘贴到新建的工作流文件中
文件名填写：`.github/workflows/build.yml`
步骤 5： 点击 "Commit changes"
步骤 6： 等待 15-20 分钟，然后在 Actions 页面下载 APK
